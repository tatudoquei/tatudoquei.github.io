O cidadão encontra pelo menos 19 barreiras para acessar dados sobre a Câmara dos Deputados e o Senado. A conclusão é de uma pesquisa feita pelo Parlametria, projeto de cientistas políticos e de dados que será disponibilizado em uma plataforma online para o público.
